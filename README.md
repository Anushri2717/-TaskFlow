# TaskFlow ✅

A simple, clean to-do list app for creating, organizing, and tracking daily tasks.

## 🚀 Features
- Add, edit, and delete tasks
- Mark tasks as complete
- Categorize/filter tasks
- Minimal, distraction-free UI

## 🛠️ Tech Stack
React, Vite, CSS (LocalStorage / Node.js + MongoDB if full-stack)

## ⚙️ Setup Instructions

1. Clone the repository
```bash
git clone <your-repo-link>
cd taskflow
```

2. Install dependencies
```bash
npm install
```

3. Run the app
```bash
npm run dev
```

4. Open `http://localhost:5173` in your browser
